## 🧰 Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js**: v22.20.0 or higher
- **pnpm**: v8.0.0 or higher (recommended) or npm/yarn as alternatives
- **Git**: For version control

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/KickButtowski80/flowpro.git
cd flowpro
```

2. **Install dependencies**

```bash
pnpm install
```

Or if you prefer npm/yarn:

```bash
# npm
npm install

# yarn
yarn install

# bun
bun install
```

### Running Locally

Start the development server on `http://localhost:3000`:

```bash
pnpm dev
```

Or with other package managers:

```bash
# npm
npm run dev

# yarn
yarn dev

# bun
bun run dev
```

The application will automatically open in your default browser.

### Building for Production

Build the application for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```

Or with other package managers:

```bash
# npm
npm run build
npm run preview

# yarn
yarn build
yarn preview

# bun
bun run build
bun run preview
```

### Complete Reset (if needed)

If you encounter dependency issues, use the nuke-and-run script:

```bash
pnpm nuke-and-run
```

This will clean all dependencies, lock files, and build artifacts, then reinstall everything fresh.
